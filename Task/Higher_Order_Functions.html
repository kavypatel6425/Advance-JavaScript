Q1. Define Higher-Order Functions.

=>A higher-order function is a function that takes another function as an argument OR Returns a function as its result, OR Does both.

Q2. Identify HOFs.

1. map()

=>Higher-Order Function

=>it takes another function as an argument and applies it to each element in an array.

2. forEach()

=>Higher-Order Function

=>it takes a function as an argument and executes it once for each array element.

3. console.log()

=>Not Higher-Order Function

=>It only prints values and does not return a function.
