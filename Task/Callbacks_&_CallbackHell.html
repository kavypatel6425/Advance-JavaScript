Q1. Theory

1. Define a callback function with a simple example.

A callback function is a function passed as an argument to another function, which is then executed after the completion of that function.

2. What is the difference between synchronous and asynchronous callbacks?

Type Synchronous Callback Asynchronous Callback
Description Executed immediately after the function that uses it finishes. Executed later, after some time (like after a task completes).
Example Array.map(), Array.forEach() setTimeout(), fetch()

3. Explain callback hell with an example.

Callback Hell occurs when multiple nested callbacks make the code hard to read and maintain.

4. List two problems that arise due to callback hell.

1.Hard to read and understand (poor code readability).

2.Difficult to debug and maintain (errors are hard to trace).
