What is Web Storage?

Web Storage is a way for websites to store data directly in the browser. It provides two main storage options:

  • localStorage
  • sessionStorage

Both allow you to store key-value pairs in the browser.

localStorage

  • Data isstored permanentlyin the browser until it is manually cleared (by code or user).
  • Even if the browser is closed and reopened, the data remains.
  • Storage limit: ~5MB (varies by browser).
